# GetIntent Status

This repository contains the status page for [GetIntent](https://getintent.ai), powered by [Upptime](https://upptime.js.org).

## Live Status

Visit [status.getintent.co](https://status.getintent.co) for the live status page.

## How it works

- GitHub Actions runs every 5 minutes to check endpoint health
- Response times and uptime percentages are tracked automatically
- Issues are created automatically when downtime is detected
- Status page is generated and deployed to GitHub Pages
